# ⚽ 2026 世界杯 · 实时比分预测

参考 [shuangseqiu666](https://github.com/wssXzzzz/shuangseqiu666) 的 **iOS 26 液态玻璃 (Liquid Glass)** 设计风格，做一个 **2026 美加墨世界杯（48 队）** 的**实时混合预测**应用。

## ✨ 核心理念：用可信度建立信任

- **首页只放次日/近期比赛预测 + 理由** — 明天就能验证，可信度高
- **冠军/16强/8强等远期预测单独成页** — 诚实承认"越远越不确定"
- 每场预测都附**数据支撑的理由**（排名、实力、状态、情境），不靠瞎猜

## 🎨 五个玻璃视图

### 🔮 预测（首页）
- **下一场大卡**：开球时间 + 预测比分 + 胜率条 + **完整预测理由**（FIFA排名对比、实力推导、近期状态、情境因素 + 一句总评）
- **近期赛程预测**：所有未踢比赛按日期分组，每场**点击展开**看预测理由
- 已踢场绿色（真实赛果）、未踢场蓝色（预测）

### 📋 小组赛
12 组，每场标绿(真实)/蓝(预测)，含积分榜（前2绿色晋级、第3橙色争最佳）

### 🔍 复盘（准确率）
**诚实的赛前预测复盘** —— 用**赛前实力**（仅 FIFA 排名 Elo，不看任何结果）逐场预测，再和真实赛果对比：
- 胜负判对率、比分命中数、场均进球误差
- 竞猜式打分：命中比分 +3、仅猜对胜负 +1
- 逐场列表：🎯命中 / ✅胜负对 / ❌未中
- **关键：预测不使用动态 Elo（避免拿"已知结果"反推），无信息泄漏**

### ⚔️ 淘汰赛
32强→决赛完整签表，每场国旗+比分，决赛金色高亮

### 🏆 夺冠
远期预测专属页（标注"越往后越不确定"）：
- 预测冠军 + 晋级之路、预测亚军
- **预测四强 / 八强 / 十六强** 名单（实力热力色块）
- 8 个晋级最佳第三名
- 夺冠概率 Top12 热力 + ECharts 各阶段晋级率 + 完整概率表

## 🔌 实时数据源

**[OpenLigaDB](https://www.openligadb.de/)** — 免费、无需 API key
- 每次访问自动抓取最新赛果，5 分钟缓存
- 顶部进度条显示「已完赛 16/72 场」
- `/api/refresh` 手动强制刷新

## 🧠 预测算法

| 环节 | 方法 |
|---|---|
| 基础实力 | FIFA 排名 → `Elo = 2050 − rank×18`（东道主+80） |
| **动态实力** | 每踢完一场，`Δ = 16×(实际净胜 − 期望净胜)` |
| 已踢比分 | 直接用 API 真实结果（绿色标记） |
| 未踢比分 | 动态 Elo → 泊松进球采样（蓝色标记） |
| **预测理由** | 排名对比 + Elo→胜率推导 + 近期战绩 + 情境(东道主/跨洲) + 总评 |
| **准确率复盘** | 赛前基础 Elo 预测比分（不看结果）→ 对比真实赛果，算胜负/比分命中率 |

## 📁 结构

```
shijiebei/
├── app.py              # Flask 路由 + 视图数据 + 实时缓存
├── predictor.py        # 动态Elo + 混合模拟 + 蒙特卡洛
├── reasons.py          # 预测理由生成器(排名/Elo/状态/情境/总评)
├── live_data.py        # OpenLigaDB 抓取 + 真实/未踢分类 + 缓存
├── wc_data.py          # 48队 + 12组 + 签表 + 球场 + 德名映射
├── templates/index.html # 单页：玻璃外壳 + 4Tab + 预测理由
├── test_predictor.py   # 核心逻辑离线单测（不触网）
└── requirements.txt
```

## 🚀 运行

```bash
pip install -r requirements.txt
python app.py        # → http://localhost:8000
python test_predictor.py   # 跑离线单测
```

## ⚠️ 说明
- 分组依据 2025-12-05 抽签；FIFA 排名为 2025-11 排名
- 未踢比赛预测仅为模型娱乐，不代表真实赛果
- 数据源：OpenLigaDB（社区维护，可能有延迟）
